# Установка пластиковых окон

## Тетовый проект

Реализовано:
- модальный окна
- отправка данных на сервер / POST запрос
- валидация формы
- калькулятор стоимости (на вкладке "балконы")
- слайдеры
- таймер
- открытие изображение в модальном окне
- плавная прокрутка к началу страницы

Инструменты:
- HTML | CSS
- Java Script
- Webpack
- Fetch
- POST
- JSON
- Git
